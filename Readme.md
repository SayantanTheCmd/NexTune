# NexTune

**NexTune** is a lightweight and efficient system optimization and module management tool designed to give users full control over NexTune-related services. It combines an intuitive, modern UI with powerful features for monitoring, managing, and configuring system modules safely and efficiently.

---

## Overview

NexTune launches a user-friendly dashboard that provides a clear view of all active modules, their status, and key system resource usage.  

The main window is organized into:

- **Module Management Panel** – View and control individual NexTune modules.  
- **Resource Monitoring Panel** – Real-time display of CPU, RAM, and GPU usage.  
- **Settings Panel** – Configure startup behavior, notifications, and module prioritization.  

The interface ensures both beginners and advanced users can easily manage NexTune modules without interfering with critical system processes.

---

## User Interface (UI)

- **Title Bar:** Displays “NexTune” with version info.  
- **Module Panel:** Shows all active NexTune modules with status indicators.  
- **Control Buttons:**  
  - **Start / Stop:** Manage individual modules or all modules simultaneously.  
  - **Refresh:** Update module statuses in real-time.  
- **Resource Panel:** Monitors CPU, RAM, and GPU usage for NexTune modules.  
- **Notification Area:** Alerts users about module errors, completions, or stoppages.  
- **Settings Icon:** Access preferences to adjust startup behavior, notifications, and module priorities.

---

## Key Features

1. **Real-Time Module Management** – Start, stop, or restart any NexTune module safely.  
2. **Resource Monitoring** – Track CPU, RAM, and GPU usage for all active NexTune modules.  
3. **Module Control Panel** – Toggle optional modules or scripts on/off and adjust priority.  
4. **Automatic Detection** – Detects all active NexTune modules on launch.  
5. **Notifications** – Alerts users about module status changes or errors.  
6. **Settings & Customization** – Configure startup behavior, notifications, and module priority.  
7. **Built-in Help & Tooltips** – Quick guidance for new users via hover-over explanations.  
8. **Safe & Efficient** – Only manages NexTune modules, avoiding critical system interference.  
9. **Portable & Lightweight** – Single executable, minimal resource consumption.

---

## Getting Started

1. **Launch NexTune** – Open the application to access the dashboard.  
2. **Monitor Modules** – Check status indicators for each module; refresh to update activity.  
3. **Manage Modules** – Start or stop individual modules or batch controls for all modules.  
4. **Adjust Settings** – Open settings to configure startup behavior, notifications, and module priorities.
